# ResumeForge AI

An AI-powered resume analyzer that helps users improve their resumes, increase ATS scores, prepare for interviews, and receive personalized career guidance.

## Planned Features

- Resume Upload
- ATS Score Analysis
- AI Resume Review
- Missing Skills Detection
- Course Recommendations
- Interview Question Generator
- Career Readiness Score
- Resume Chat Assistant

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS

Backend:
- FastAPI

AI:
- Google Gemini

Deployment:
- Vercel
- Render
